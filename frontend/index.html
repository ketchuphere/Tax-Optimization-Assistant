Architecture
1
Data Collection & Vectorization
Financial data is structured and normalized. ChromaDB with sentence-transformers stores anonymized profiles for efficient similarity lookup using cosine distance.
2
RAG Retrieval
Top-k most similar profiles are retrieved using income, filing status, and deduction pattern similarity — forming the RAG context window.
3
LLM Prompt Engineering
Retrieved profiles + a tax regulation knowledge base are injected into a structured prompt. Claude generates JSON-structured personalized recommendations.
4
Personalized Dashboard
Recommendations are ranked by priority and potential savings, with interactive visualizations and a conversational AI for follow-up questions.
